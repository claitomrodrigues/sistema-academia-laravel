# 💪 Sistema de Academia

Sistema web desenvolvido para gerenciamento de alunos, planos e treinos de uma academia.

## 📌 Sobre o projeto

Este sistema tem como objetivo facilitar a gestão de academias, permitindo o controle de alunos, planos de pagamento e fichas de treino de forma organizada e eficiente.

## 🚀 Funcionalidades

* Cadastro de alunos (CRUD completo)
* Gestão de planos (valores e descrição)
* Cadastro de exercícios
* Montagem de fichas de treino
* Controle de mensalidades
* Listagem de pagamentos

## 🛠 Tecnologias utilizadas

* PHP
* MySQL
* Bootstrap
* HTML/CSS

## 🗄 Estrutura do sistema

### Entidades principais:

* **Aluno**: nome, CPF, data de nascimento, objetivo
* **Plano**: nome, valor, duração
* **Matrícula**: vínculo entre aluno e plano
* **Exercício**: nome, grupo muscular
* **Treino**: ficha personalizada do aluno

## ▶️ Como executar

1. Clone o repositório:

```
git clone https://github.com/seu-usuario/sistema-academia.git
```

2. Coloque o projeto no XAMPP ou Laragon

3. Crie o banco de dados no MySQL

4. Importe o arquivo `.sql` (se houver)

5. Configure a conexão com o banco no projeto

6. Acesse no navegador:

```
http://localhost/sistema-academia
```
